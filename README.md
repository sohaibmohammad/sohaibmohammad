## 🚀 Featured Projects

### 🛒 [Ecommerce-Backend](https://github.com/sohaibmohammad/Ecommerce_Backend)
> **Scalable E-commerce API built with .NET & EF Core**
* 🏗️ **Architecture:** Built following **Clean Architecture** patterns to enforce strict separation of concerns.
* 📦 **Core Features:** Robust full CRUD for product catalogs, categorized browsing, and dynamic shopping cart tracking.
* 💾 **Data Layer:** Managed persistent relational data workflows and optimized SQL transactions using **Entity Framework Core**.

### 💼 [JobPlatform-Backend](https://github.com/sohaibmohammad/Backend_JobPlatform)
> **Full-Stack Recruitment & Application System**
* 🔐 **Security:** Secure identity workflows using custom **JWT Authentication & Role-Based Authorization**.
* ⚡ **Real-Time Integration:** Embedded **SignalR** infrastructure for handling notification dispatching and instant application state updates.

### 💬 [RealTimeChat-Backend](https://github.com/sohaibmohammad/realTimeChat--Backend)
> **Instant Messaging Engine utilizing WebSockets**
* 🔄 **Sync Engines:** Leveraged **SignalR Hubs** to achieve sub-second data synchronization between participants.
* 📊 **State Management:** Designed highly relational database schemas focused on tracking message deliveries, history, and active sessions.
